	How to use demo scenes?

		1) Add new resolution to the Game tab.
			1.1) Click on resolution list in Game tab.
			1.2) Click on the small plus button.

				- Click on OK button.
			1.4) Press OK button.
		2) Change resolution of Game tab to 960x600px.
		3) Add all demo scenes to Build Settings dialog.
			3.1) Press Control+Shift+B (Windows) or Command+Shift+B (Mac).
			3.2) Or use menu select "Top Menu->Edit->Build Settings".
			3.3) In Project tab, look for a folder names "GUI Animator/Demo (CSharp)/Scenes".
			3.4) select Demo00 to Demo 08 files in "GUI Animator/Demo (CSharp)/Scenes" to "Scene In Build" in "Build Settings" dialog.
			3.5) Close "Build Settings" dialog.
		4) In Project tab, double click on "GUI Animator/Demo (CSharp)/Scenes/Demo00 (960x600px)" file or open any scene as you like.
		5) Press Play button.

	How to switch tweeners?

		GUI Animator uses preprocessor directive to support tweeners.
		1) Open Build Settings dialog.
			1.1) Press Control+Shift+B (Windows) or Command+Shift+B (Mac).
			1.2) Or use menu select "Top Menu->Edit->Build Settings".
		2) Click on "Player Settings" button.
		3) In Inspector tab, look at "Configuration->Scripting Define Symbols" text area. Note you can assign only one tweener at a time.
			3.1) DOTween, type DOTWEEN in the text area.
			3.2) HOTween, type HOTWEEN in the text area.
			3.3) iTween, type ITWEEN in the text area.
			3.4) LeanTween, type ITWEEN in the text area.
			3.5) If you leave it blank GUI Animator will use GETween as default. Note GETween is default tweener.

		**preprocessor directives are lines included in the code of programs preceded by a hash sign (#). These lines are not program statements but directives for the preprocessor. The preprocessor examines the code before actual compilation of code begins and resolves all these directives before any code is actually generated by regular statements.

	Designer's workflow?

		1) Create new scene.
		2) Add UI objects and arrange them.
			Note if you are new for Unity UI please visit,
			- https://unity3d.com/learn/tutorials/modules/beginner/ui
			- http://docs.unity3d.com/Manual/UISystem.html
			- http://docs.unity3d.com/ScriptReference/UI.AnimationTriggers.html
		3) Drag GUIAnimSystem from Prefabs folder (in Project tab) and drop it into Hierarchy tab.
		4) Drag GEAnim script from Scripts folder (in Project tab) and drop it into UI objects that you what to animate.
			Note to save your time, you can select multiple UI objects in Hierarchy tab then you drop GEAnim script to the Inspector tab.
		5) Setup GEAnim's parameters.
		6) Press Play button to watch the result or repeat step 5th.
			Using Animation Mode of GEAnimSystem will let you filter and play specific In/Idle/Out animations.
		7) Select GUIAnimSystem in Hierarchy tab and disable the Auto Animation.
		8) Create a script to control the scene.

	How to setup a new scene with GUI Animator?

		1) Create new scene in Unity Editor.
		2) In Project tab, drag GEAnimSystem in "GUI Animator/Prefabs" folder and drop it on empty area of the Scene tab or Hierarchy tab.
		3) Create a UI element, for example we add Image. Note Unity automatically creates a Canvas and EvenSystem objects for UI Image when the first UI element is added to the scene.
		4) In Project tab, select GEAnim in Scripts folde. Drop it into Image in the Hierarchy tab.
		5) In Hierarchy tab, select Image you should see a component names GEAnim in Inspector tab.
		6) In GEAnim component, expand Move In node and make change to some parameters.
		7) Press Play button.
			7.1) You should see Image moves from the top-edge of screen.
			7.2) The animation delays for 3 seconds before it takes 2 seconds to finish the animation.
			7.3) If you are new on Unity or you do not familiar with concept of tweeners, you may need some time to give it a try to understand it.
			Note there are 3 types of animations; In, Idle, Out. Animations play automatically when you press Play button.
		8) In Hierarchy tab, select GEAnimSystem. You should see GEAnimSystem component in Inspector tab.
		9) GEAnimSystem contains 4 parameters;
			9.1) GUI Speed; this is amount of global speed to multiply with the speed of each GEAnim in the scene. You can use GUI Speed to control all UI elements' speed in run-time via script. We will talk about it in another section.
			9.2) Auto Animation; if true, animation play automatically when scene starts. Otherwise, you can use scripts to control all animations.
			9.3) Animation Mode; this parameter only works when Auto Animation (9.2) is true, you can select what type of animation (In/Idle/Out) to play when you press Play button.
			9.4) Idle Time; amount of time to play Idle animation. This parameter only works when Auto Animation (9.2) is true.

	How to control GUI Animator by scripts?

		1) In Project tab, right click and select create a C# script file them name it as you like. For example we name it MyGEAnimScript. Note script file should be in a folder names Scripts.

		2) Double click MyGEAnimScript Unity will open your script editor.
	
		3) Insert these line below "using UnityEngine" statement.
	
			// This line enables Unity UI elements in this script.
			using UnityEngine.UI;	

		4) In Awake function add these lines.
	
			//////////////////////////////////////////////////////////////////////
			// If GEAnimSystem.Instance.m_AutoAnimation is false,
			// all GEAnim elements in the scene will be controlled by scripts.
			//////////////////////////////////////////////////////////////////////
			if (enabled)
			{
				// Change GUI speed to faster or slower than the actual.
				GEAnimSystem.Instance.m_GUISpeed = 1.0f;

				// Lets you control all animations via script.
				GEAnimSystem.Instance.m_AutoAnimation = false;
			}

		5) Add GEAnim variables.

			// You should name them as you like
			public GEAnim m_MyImage = null;
			public GEAnim m_MyButton = null;
			public GEAnim m_MyText = null;
			public GEAnim m_MyPanel = null;

		6) Back to Unity Editor, In Hierarchy tab create an empty object and name it MyGEAnimController.
	
		7) Drag MyGEAnimScript (in Project tab) to MyGEAnimController (In Hierarchy tab).

		8) Select MyGEAnimController, you should see MyGEAnimScript component in Inspector Tab.

		9) Bind UI elements to MyGEAnimScript parameter.
	
			9.1) In Hierarchy tab, select MyGEAnimController.
			9.2) Lock the Inspector tab (click on a lock icon at the top right of Inspector tab while MyGEAnimController is selecting).
			9.3) Drag each UI element that contans GEAnim and drop it into GEAnim parameters in MyGEAnimScript component.

		10) Switch back to script editor, now you are ready to control them.

			10.1) To Play In animation.

				A) Play In animations of m_MyImage and also play animations of its children.

					GEAnimSystem.Instance.MoveIn(m_MyImage.transform, true);

				B) Play In animations of m_MyImage but ignore animations of its children.

					GEAnimSystem.Instance.MoveIn(m_MyImage.transform, false);

				C) Same A.

					m_MyImage.MoveIn();

				D) Same B.

					m_MyImage.MoveIn(eGUIMove.Self);

				E) Play In animations of m_MyImage's children.

					m_MyImage.MoveIn(eGUIMove.Children);

			10.2 To Play Out animation.

				A) Play Out animations of m_MyImage and also play animations of its children.

					GEAnimSystem.Instance.MoveOut(m_MyImage.transform, true);

				B) Play Out animations of m_MyImage but ignore animations of its children.

					GEAnimSystem.Instance.MoveOut(m_MyImage.transform, false);

				C) Same A.

					m_MyImage.MoveOut();

				D) Same B.

					m_MyImage.MoveOut(eGUIMove.Self);

				E) Play Out animations of m_MyImage's children.

					m_MyImage.MoveOut(eGUIMove.Children);

			10.3) Utilities GUI Animator contains some useful methods that you may need them.

				// Load next level with a delay
				GEAnimSystem.Instance.LoadLevel(string LevelName, float Delay);

				// This function enables/disables Graphic Raycaster. Note Graphic Raycaster let Canvas responds or ignores to the user input.
				GEAnimSystem.Instance.SetGraphicRaycasterEnable(GameObject gob, bool Enable);
				GEAnimSystem.Instance.SetGraphicRaycasterEnable(Canvas cav, bool Enable);
				GEAnimSystem.Instance.SetGraphicRaycasterEnable(Transfrom trans, bool Enable);

				// Enable/disable a Button or many Buttons, depends on the hierachy of the given transform.
				GEAnimSystem.Instance.EnableButton(Transform trans, bool Enable);

				// Play a ParticleSystem or many ParticleSystems, depends on the heirarchy of the given transfrom.
				GEAnimSystem.Instance.PlayParticle(Transform trans);

				// Stop a ParticleSystem or many ParticleSystems, depends on the heirarchy of the given transfrom.
				GEAnimSystem.Instance.StopParticle(Transform trans);

				// Keep a ParticleSystem or many ParticleSystems be alive, depends on the heirarchy of the given transfrom.
				GEAnimSystem.Instance.DontDestroyParticleWhenLoadNewScene(this.transform, true);

	How to replace "GUI Animator FREE" with "GUI Animator for Unity UI"?

		1) Make sure your project has "GUI Animator FREE" and "GUI Animator for Unity UI".

		2) Replace GUI Animator FREE components in demo scenes.
			2.1) Open a demo scene of UI Pack. For example, we will use Demo 01 - Landscape - Intro & Loading scene.
			2.2) In Project tab, select GUIAnimSystem prefab.
			2.3) Drag and drop it into empty space in Hierarchy tab.
			2.4) Move children objects of GUIAnimSystemFREE to GUIAnimSystem.
			2.5) Remove GUIAnimSystemFREE.
			2.6) In Hierarchy tab, look for search text box.
			2.7) Type GUIAnimFREE in search text box. The Hierarchy tab will show only the objects that contain GUIAnimFREE component.
			2.8) Select the filtered objects in Hierarchy tab.
			2.9) Drag GEAnim script file from Project tab.
			2.10) Drop it into Script parameter in Inspector tab. The component will change to GEAnim.
				Note GEAnim component can be switched between Legacy and Friendly Inspector modes. First one supports Multi-object editing but it looks not friendly unlike another.
			2.11) Repeat step 2.1 to 2.11 for any scene you want.

		3) Replace GUI Animator FREE components for demo scripts.
			3.1) In Project tab, double click Demo01_IntroAndLoading script file to open it (script files are in UI Pack/Toony/Scripts folder). For example we will use MonoDevelop.
			3.2) Press Command-Shift-F for Mac or Control-Shift-F for Windows. Replace in Files dialog will appear.
			3.3) Replace GUIAnimSystemFREE.ePosMove in all files with ePosMove.
				- Click Replace in Files button.
				- Type GUIAnimSystemFREE.ePosMove in Find text box.
				- Type ePosMove in Replace textbox.
				- Press Replace button.
			3.4) Replace GUIAnimFREE.ePosMove with ePosMove.
				- Click Replace in Files button.
				- Type GUIAnimFREE.ePosMove in Find text box.
				- Type ePosMove in Replace textbox.
				- Press Replace button.
			3.5) Replace GUIAnimSystemFREE with GEAnimSystem.
				- Click Replace in Files button.
				- Type GUIAnimSystemFREE in Find text box.
				- Type GEAnimSystem in Replace textbox.
				- Press Replace button.
			3.6) Replace GUIAnimFREE in all files with GEAnim.
				- Click Replace in Files button.
				- Type GUIAnimFREE in Find text box.
				- Type GEAnim in Replace textbox.
				- Press Replace button.
